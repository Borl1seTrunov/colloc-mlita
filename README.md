# Коллоквиум по дисциплине МЛиТА

## Инструкция по запуску программы
- склонировать репозиторий 

     ```
     git clone https://github.com/rersad1/coloc-mlita.git
     ```
- установить виртуальное окружение python
    - Windows:  
    
    ```
    python -m venv .venv
    ```

    - MacOS/Linux: 
    
    ```
    python3 -m venv .venv
    ```
- активировать виртуальное окружение python
    - Для Windows: 
    
    ```
    .venv\Scripts\Activate.ps1
    ```

    - Для MacOS/Linux: 
    
    ```
    source .venv/bin/activate
    ```
    - выйти из окружения можно с помощью 
    
    ```
    deactivate
    ```
- установить все необходимые зависимости
    - Windows: 
    
    ```
    pip install -r requirements.txt
    ```

    - MacOS/Linux: 
    
    ```
    pip3 install -r requirements.txt
    ```
- запустить программу
    - Windows: 
    
    ```
    python main.py
    ```
    - MacOS/Linux: 
    
    ```
    python3 main.py
    ```
